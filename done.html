  • Problem: Embedded Google Form dropdown displays fucked up on mobile, but removing the embedded form entirely poses an accessibility issue.
    Solution: Add #content-desktop {display: block;} #content-mobile {display: none;} @media screen and (max-width: 768px) { #content-desktop {display: none;} #content-mobile {display: block;} }to CSS, wrap embed in content-desktop div.
  • Learned to properly style pseudo-classes of specific link classes. For example, to style the hover state of all links with a class of "new", write selector as:
    a.new:hover {}
    To select the hover state of all links with a class of "new" with a parent class of "main":
    .main a.new:hover {}
